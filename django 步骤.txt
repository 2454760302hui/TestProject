1.  新建virtualenv 虚拟环境 pip install virtualenv

2.当前目录输入dir 进入Scripst 目录下 运行activate.bat  【前面展示testvir】说明 成功

3.退出虚拟环境 运行  deactivate.bat 

4.另一种在windows下的按照方式 pip install virtualenvwrapper-win 【和上一种的区别：不需要运行虚拟环境，可以直接进入虚拟环境】

5.创建新项目mkvirtualenv testvir2 

6.退出虚拟环境 deactivate

7.查看当前有哪些虚拟环境  workon

8.进入testvir2虚拟环境 workon testvir2

9.查看当前的开发包  pip list

10.新建django项目-在base interpreter中从env中选择python.exe应用程序，然后创建，成功后，在该项目下展示__init__,settings urls,wsgi 等 说明成功
